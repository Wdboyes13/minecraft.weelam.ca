Mod Suggestions

Mod Suggestions are accepted at the #mod-suggestions Discord channel

Rules

Any user creates a poll, with options "Yes" and "No" If the mod is not well known, they must provide a link The mod name should be the title of the poll (link can be sent after) A mod vote win does not neccesarily mean it will be added A mod vote must have a majority of the players to be considered When a mod gets voted, owners will review it to see if it will negatively or positively affect the server, including performance If the mod is unavailable for the server, it is automatically disqualified As well as if the Mod / Developer is found to be involved in any illegal activities, it will be disqualified, there is 0 exceptions here Mod count exceptions may be applied if the mod is seen as important enough
